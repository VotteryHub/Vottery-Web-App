## 2. Recurring themes (cross-file)

- **Core product:** Vottery = Facebook-like social + E2E encrypted, verifiable, auditable, gamified elections; subscription for creation/participation; monetization via ads (participatory), subscriptions, participation fees.
- **Elections & Voting:** Create / Vote / Verify / Audit (left menu + header dropdown); voting types Plurality, Ranked Choice, Approval; optional MCQ; gamified = vote = lottery ticket; 3D slot machine for draws; prize types (monetary, non-monetary, projected revenue); multiple lucky winners by %; unique election ID, URL, QR; 8 regional zones for fees.
- **Admin:** Dashboard, User/Voting/Revenue/Deposit/Withdraw management; disable/enable **all features**, **countries**, **integrations** (with cost limits); content distribution % (election vs social); participation fee global + country-wise; biometric country-wise; admin roles; real-time analytics.
- **Creator:** 70/30 (or configurable per country); payout automation; Stripe Connect; tax (W-9/W-8BEN); creator analytics; onboarding; marketplace; brand partnerships; success academy.
- **Participatory Ads:** Clone Facebook ads mechanism; “Ad Is The Vote”; Market Research / Hype / CSR; CPE/CPX; 1 ad per 7 organic; Shaped → Supabase+OpenAI; Ad Slot Manager so internal ads first, AdSense only unfilled slots.
- **Gamification:** VP (Vottery Points); levels, badges, challenges, leaderboards, streaks; redemption shop; prediction pools (Brier); gamified feeds and ad rewards; platform-wide monthly gamification (percentization by country/segment).
- **Integrations:** Supabase, Stripe, Resend, Twilio, Google Analytics, AdSense, OpenAI, Anthropic, Perplexity, (Shaped replaced by Supabase+OpenAI), Datadog, Sentry, Slack.
- **Mobile:** Same features as web where applicable; Flutter app (separate codebase); touch-friendly admin; offline (Hive); push notifications; performance (lazy load, &lt;2s).
- **Security & compliance:** E2E, RSA, signatures, hashing, ZKP; fraud detection (OpenAI/Claude/Perplexity); threshold alerting; audit logs; GDPR/CCPA; VVSG 2.0; WCAG 2.1.
- **Docs:** Technical platform documentation; development methodology guide; full platform documentation (PDF-friendly).
