# Vottery V1 UI/UX Specification

## 1. Design Tokens & Aesthetics
- **Style**: Glassmorphism, premium dark mode, vibrant gradients.
- **Typography**: Inter / Outfit (Google Fonts).
- **Interactions**: Smooth micro-animations (Framer Motion), haptic-like visual feedback.

## 2. Core Layouts
- **AppShell**: Responsive sidebar (desktop) -> Bottom navigation (mobile).
- **PageContainer**: Centered content grid with `lg` breakpoint gating for sidebar elements.

## 3. Key Components
- **Carousel Orchestrator**: Supports 2D snap, vertical stack, and smooth gradient flows.
- **QR Reference**: Standardized Vottery QR codes for election sharing.
- **Slot Machine Reference**: Gamified voting results (3D animations).
- **Vottery Logo**: Premium SVG assets only.

## 4. Navigation Structure
- **Categories**: Politics, Tech, Entertainment, Sports, Social, Finance, Local.
- **Search**: Advanced filters with predictive results (Shaped AI).
